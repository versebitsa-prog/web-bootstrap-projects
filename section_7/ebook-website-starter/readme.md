## Installation & Usage

To set up the project locally:

1-Install Node.js

2-Clone this repository

## Install dependencies:

npm install


## Compile Sass to CSS:

npm run sass:build


## Or watch Sass files for changes (auto compile):

npm run sass:watch 

##### #### ## 

Note: Do not edit the Bootstrap source files directly. Use styles.scss for your custom styles.
