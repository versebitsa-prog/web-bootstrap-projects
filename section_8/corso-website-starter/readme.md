This is a responsive Bootstrap website I built for showcasing training courses, workshops, seminars, and more.
I designed a modern layout with both light and dark themes, customized colors and styles, and implemented interactive components to improve user experience.

<img src="./images/screen.png" />

## Features

Responsive layout for desktop and mobile

Sticky navbar with dynamic style changes on scroll

Carousel image slider for featured courses

Registration and email subscription forms

Font Awesome icons for visual enhancements

Custom colors, backgrounds, and styles using Sass

All custom styles are in styles.scss.

## Installation & Usage

To set up the project locally:

1- Install Node.js

2- Clone this repository

## Install dependencies:

npm install


## Compile Sass to CSS:

npm run sass:build


## Or watch Sass files for changes (auto compile):

npm run sass:watch


## Note: Do not edit Bootstrap source files directly. Use styles.scss for custom styles and modifications.
