## Features

Dark-themed design with bright buttons & text for contrast

Responsive layout for desktop and mobile

Rolling words animation with custom script

Navbar scroll effect

Video modal with animated play button

Pricing grid for services

Styled modals for projects and details

Font Awesome icons

All custom styles implemented using Sass (styles.scss)

## Usage
To set up the project locally:

1- Install Node.js

2-Clone this repository:

## Install dependencies:

npm install


## Compile Sass to CSS:

npm run sass:build


Or watch Sass files for changes (auto compile):

npm run sass:watch


## Note: Do not edit Bootstrap source files directly. Use styles.scss for your custom styles and modifications.
