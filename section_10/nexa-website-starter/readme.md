## Usage

To set up the project locally:

Install Node.js

Clone this repository:

git clone https://github.com/your-username/yavin-website.git


Install dependencies:

1- npm install


Compile Sass to CSS:

 2- npm run sass:build


Or watch Sass files for changes (auto compile):

npm run sass:watch


## Note: Do not edit Bootstrap source files directly. Use styles.scss for custom styles and modifications.
